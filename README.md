# check-driving-exam-times

Simple script to check available times theorical and practical exams on https://fp.trafikverket.se

## Setup

Set this environment variables:

* SOCIAL_SECURITY_NUMBER
* TELEGRAM_TOKEN
* TELEGRAM_CHAT_ID

Install using this command:

```bash
npm i
```

## Run

Run only one time:

```bash
node check-time.js
```

Run forever:

```bash
bash run.sh
```
